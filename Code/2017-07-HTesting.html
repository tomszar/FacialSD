Hypothesis testing

In this script we will see the preliminary hypothesis testing on the effects of sex on face shape, as well as the allometric and non-allometric decomposition of sexual dimorphism. Because this is a sample testing, we will take a random sample of 200 males and 200 females to ensure a fast computation of these tests.

Facial sexual dimorphism analysis

Differences in location

I will first test for differences in location between males in females in the face PCA. To do that I will use two different methods. In both methods I will use the full set of PCs, and a subset of the first 10 PCs. First, I will compare males and females location using Hotellings T2 statistic, which is a multivariate t-test. The downside is that we cannot control for other covariates, such as Age and BMI. The results are virtually identical when using a normalized PC space (not shown here).

Second, I will use the more involved PERMANOVA (Anderson 2001) using the mahalanobis distance between males and females. The benefit of mahalanobis distance is that the within-group variance is taken into account, and not only the euclidean distance in space.

Differences in dispersion

I will use Anderson (2006) method for testing differences in dispersion between sexes

dist  <- vegdist(coeffs200[, 7:16], method = "mahalanobis")

plotmeans <- function(matrix, means, factor, x1, y1)
{
  p1 <- ggplot(matrix, aes_string(x = x1, y = y1)) +
            geom_point(aes(colour = factor), alpha=0.1) +
            geom_point(data = means, aes_string(x = x1, y = y1), color="red", size=3) +
            theme_pubr()
  p1 <- ggpar(p1, palette = "jco")
  return(p1)
}

To obtain the allometric component, we use the previous multivariate linear regression. To decompose the total FSD into its allometric and non-allometric components, we use the estimated effect of height on the face, scaled by the mean difference between females and males.

consensus[3,] <- model$coefficients[2,]
consensus[4,] <- coef(model)[5,] * (mean(coeffs200[coeffs200$Sex == 'Male',4]) -
                                    mean(coeffs200[coeffs200$Sex == 'Female',4]))
consensus[5,] <- as.numeric(consensus[3,]) - consensus[4,]
row.names(consensus) <- c("Males", "Females", "TotalSD", "Allometric", "Non-allometric")
